# Бот для расчета таможенных пошлин на легковые автомобили

## Описание
Telegram-бот, который рассчитывает российские таможенные пошлины для импортируемых легковых автомобилей в 2026 году. Бот предназначен для физических и юридических лиц, поддерживает несколько валют и фокусируется на легковых автомобилях.

## Особенности
- Расчет таможенных пошлин для автомобилей возрастом 1-3 года: 48% от стоимости + утилизационный сбор
- Для автомобилей возрастом 3-5 лет: фиксированная пошлина, зависящая от объема двигателя + утилизационный сбор
- Поддержка нескольких валют (USD, CNY, KRW)
- Различные расчеты для физических и юридических лиц
- Интерактивное меню на русском языке

## Установка и запуск

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env` и добавьте токен вашего Telegram-бота:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

3. Запустите бота:
```bash
python bot.py
```

## Логика расчета

### Для автомобилей возрастом 1-3 года:
- Таможенная пошлина: 48% от стоимости покупки (инвойса)
- Утилизационный сбор: зависит от количества лошадиных сил
  - До 160 л.с.: фиксированная сумма
  - Свыше 160 л.с.: увеличенный сбор, зависящий от мощности

### Для автомобилей возрастом 3-5 лет:
- Фиксированная таможенная пошлина: зависит от объема двигателя
- Утилизационный сбор:
  - До 160 л.с.: фиксированная сумма 5200 руб.
  - Свыше 160 л.с.: зависит от объема двигателя и количества лошадиных сил

## Использование
1. Отправьте команду `/start` боту в Telegram
2. Нажмите кнопку "Рассчитать таможню"
3. Введите стоимость автомобиля
4. Выберите валюту
5. Введите дату производства автомобиля
6. Введите объем двигателя
7. Введите количество лошадиных сил
8. Выберите тип импортера (физическое или юридическое лицо)
9. Получите результаты расчета

## Структура проекта
- `bot.py` - основной файл с логикой бота
- `requirements.txt` - зависимости проекта
- `.env` - файл с конфиденциальными данными (токен бота)

## Примечания
- Курсы обмена валют являются условными и должны быть обновлены актуальными значениями из внешнего API
- Расчеты основаны на действующих в 2026 году правилах таможенного регулирования
- Бот не хранит персональные данные пользователей
